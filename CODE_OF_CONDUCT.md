# Code of Conduct

`zod-nest` adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. The full text lives at the link above.

## In short

- Be respectful in issues, pull requests, and discussions.
- Assume good intent. People come from different backgrounds and have different communication styles.
- Disagree with ideas, not with people. Technical pushback on a proposal is welcome; personal pushback isn't.
- Keep the discussion focused on the work — the library, the bug, the feature.

## Scope

This code applies anywhere the project's community space extends: the GitHub issue tracker, pull requests, discussions, commit messages, and any other forum (Slack, Discord, conference talks) where someone is participating as part of the `zod-nest` community.

## Reporting

If you experience or witness behaviour that violates the Contributor Covenant in any `zod-nest` space, raise it on GitHub and tag a repository maintainer — in an issue, a PR comment, or a discussion thread. The maintainers will respond there.

## Enforcement

The Contributor Covenant's standard enforcement ladder (correction → warning → temporary ban → permanent ban) applies. Decisions are at the maintainer's discretion, in line with the published guidelines.
